我说的不一定对-梁欢-阅读记录-3

发表于 2017-03-20

1

人们会对身边很多习以为常的事情表现出轻慢的态度。比如对亲人没有耐心，而对陌生人彬彬有礼。对没有深入了解过的东西表示不屑。正如段子PHP是世界上最好的语言，黑的多了，竟真的有人这么以为。似乎很少对常见的东西深入了解，见的多了，好奇心也没了。好奇心没了还是比较好的情况，更严重的是对常见的东西看轻，表达一种不屑的态度。这实在是糟糕。

2

游戏是学习的终极方式。

游戏是艺术的终极形态。

以上是梁欢对游戏的两个预言。从这部分内容来说，梁欢从游戏中学习到了很多知识，甚至游戏让他走出抑郁。说实话，他提到的那些游戏我小时候听都没听过。不知道他的抑郁症在买电脑的行为中产生了多大的作用，单单就买电脑来说，我小时候也很难负担得起。特别是要负担得起一款大型游戏的电脑。我小时候连小霸王都没有，唯一的游戏机是那种俄罗斯方块的小机子。直到后来，家里买了DVD还是VCD来着，这只是个前提，在拥有了播放机之后的某一天，我偶然知道了光盘游戏，就是可以放入光盘，然后在机器后面接上游戏手柄就可以玩游戏了。我兴冲冲的买回家玩了之后，我发现没有多少性质了。不知道是我的年纪变大，还是相对于当时接触的其他游戏，如QQ堂，光盘游戏让我实在提不起兴致。身边环境是相当重要的，如梁欢接触到《魔神战记2》、《大航海时代4》、《NBA Live 2001》、《文明3》等游戏，这些游戏确实能让他学到很多知识。但是对于没有条件的人来说，可能甚至不知道电子游戏是个什么东西。另外，所谓的“电子海洛因”，我个人感觉，更可能是指一些毫无意义的游戏，我指的毫无意义是除了打发时间，回过头来，除了感叹一下当初的情怀之外，没有别的收获了。当然也或许是我玩的不够深入。另外，正如梁欢提到的，游戏的水平由制作这款游戏的人的智识水平决定。这在现在看来，通俗的，不那么准确的说，就是收费游戏的质量要比免费游戏好许多。如前文所述的“支持正版”那部分内容。在我看来，对于大部分泡在网吧的学生来说，网络游戏的后果，确实是称得上“电子海洛因”了。

4

好好说话2.0。常常读上几遍十分有好处。
抄录如下：

  1. 不使用脏话。即便脏话用于表达程度之强烈，也应尽量慎用。绝大多数情况下，它只会让人感觉使用者缺乏足够的遣词造句的能力。
  2. 讨论的目的，在于「寻找什么是对的」，而非「证明自己是对的」。后者也是导致「拒不认错」这个行为的原因。
  3. 讽刺是一种高等级的、令人激赏的表达技巧，但它的效力只作用于合适的语境下。最初发表观点时，可以讽刺（包括但不限于搞笑、调侃、吐槽），这有助于观点的传播；但与人进入具体讨论时，则要避免讽刺。
  4. 不做「动机揣测」。以下皆为动机揣测：
  • 「你想红想疯了。」
  • 「大家散了吧，此人已经被收买。」
  • 「不就是借机抱大腿，炒作自己吗？」
  1. 不使用「资格论」。以下皆为资格论：
  • 「你懂个屁！」
  • 「你一个xxxx，有什么资格评论别人？」
  • 「等你也能xxxx，再来跟我说吧！」
  1. 不「混淆概念」。以下皆为混淆概念：
  • 「xxxx假唱又怎么样？毕竟他们还未成年。」
  • 「xxxx的产品好用？他们创始人是同性恋！」
  • 「反对婚姻制度？那你是怎么被生出来的？」
  1. 尽可能少的使用反问句，多使用陈述句。尤其要避免「反问句」+「呵呵」句式，考虑到该句式已成为当今网络世界最大争吵源之一，且它暗示着使用者「我根本不打算进行任何有效讨论，我就是单纯要鄙视你」的立场，我们应当像不随地吐痰一样，不使用该句式。
  2. 当自己的观点、论据、逻辑遭受对方质疑时，相信对方是「对事不对人」的，因为你对对方也是如此；倘若发现对方不是如此，则宁可直接且优雅的快速退出讨论，也不使自己陷入同等「对人不对事」的可悲境遇。
  3. 在讨论过程中向对方表达感谢，有助于维持一段良性讨论的氛围；在讨论结束之后向对方表示感谢，有助于开启或增温一段友情。一个奇怪的现象是，人们在网络世界里往往不太习惯向他人表达感谢，这时候如果你能勇于致谢，你与被致谢方都能收获额外的愉悦。
  4. 诚恳承认自己的错误，并始终为自己的错误言行及其引发的后果负责。这是最重要的。

5

这部分内容描述了一个群体是如何从兴起到衰退。有很多感同身受的内容。只是梁欢提出的那些规则确实是不容易做到，不是指规则不容易做到，而是指能够寻求到有着这些规则的默契的人很难。这点我觉得我和梁欢有差不多的想法，这种规则如果要我去说出来的话，这个规则就没有意义了。之所以不能说出这个规则，就是为了让群体中的人自身就具备这种素质，从而自然而然地组成一个群体，这样就不会对规则产生拒绝的想法，也不会被规则所约束强迫，因为这是这个群体中的个体天生自带的，无需额外强求。
