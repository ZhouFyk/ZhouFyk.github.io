PHP manual-mysqli-Multiple Statements-翻译

发表于 2017-03-25

MySQL允许在一个语句字符串中使用多个语句。 一次发送多个语句可以减少客户端 - 服务器的往返行程，但需要特殊处理。

必须使用mysqli_multi_query()执行多个语句或多个查询。 语句字符串的各个语句用分号分隔。 然后，执行语句返回的所有结果集都必须被取出。

MySQL服务器允许多重语句中有的语句返回结果集而有的语句不返回结果集。

安全考虑
API函数mysqli_query()和mysqli_real_query()不设置在服务器中激活多个查询所需的连接标志。 多个语句使用了额外的API调用，以减少意外的SQL注入攻击的可能性。 攻击者可能会尝试添加如 DROP DATABASE mysql or; SELECT SLEEP(999)。 如果攻击者成功地将SQL添加到语句字符串，但是没有使用mysqli_multi_query，则服务器将不会执行第二个注入和恶意的SQL语句。
