  1. 为了防止 url 存在问题，我想要把不符合要求的 url 都指向首页。然而没有搜到相关的内容。看了几遍文档，Laravel HTTP 路由功能：

正则表达式约束#
你可以使用 where 方法来规范你的路由参数格式。

使用正则表达式来解决：

Route::get('{error_url}',function(){
        return redirect('/');
    })->where('error_url','[\w\W]+');

把该路由放在最后，这样之前都没匹配到的就是错误的路由，那么就重定向到首页就可以了。
