接口文档

  • app : api function，接口类型（为哪个渠道提供）：接口功能简述。
  • url：请求的 URL
  • request method： 请求方式
  • params：请求参数
  • format：请求的格式
  • return：返回值
    • code ：错误码
    • msg ：错误信息
    • data ：数据
  • example：请求示例
