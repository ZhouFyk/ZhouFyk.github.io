而 tcp 与 unix 的差别在哪里呢。

nginx和php-fpm的通信方式有两种，一种是TCP的方式，一种是unix socket方式，哪种好一点？ :

  1. unix socket适合nginx和php-fpm都在一台机器上
  2. TCP适合nginx和php-fpm不在同一台机器上面
  3. TCP通信还需要三次握手
  4. 生产环境，考虑高可用的话。会有多台nginx搭配多台php-fpm使用
  5. unix socket比套接字更快，因为其中没有套接字的路由操作，开销会比套接字更小。
  6. tcp协议能保证数据的正确性，sock不能保证。
