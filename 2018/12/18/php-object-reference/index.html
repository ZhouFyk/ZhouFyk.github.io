看到两个示例的结果，可以很清楚得理解官方文档了。

赋值，是两个变量保存着一个标识符的拷贝。所以如果 $b = $c，那么 $b 保存的与 $a 相同的标识符就改成了与 $c 保存的标识符相同。而为什么修改内容之后，两个变量输出都会变，因为双方保存的标识符指向的内容被改变，而标识符未变，所以表示出来也是相同的，都被改变。所以赋值改变变量保存的标识符。

引用，是两个变量指向同一个内容。在示例中可以看到，$b = $c 时，$a 也改变了，因为 $b / $a 指向同一个内容。所以三个变量的内容是相同的。如果改变 $c 的值，发现 $a / $b 是相同的，而 $c 的值变了。
